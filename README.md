# ClinicFlow

MVP SaaS web app for small dental clinics to reduce missed appointments.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- PostgreSQL
- Prisma
- Credentials-based staff authentication

## MVP scope

- Clinic owner and staff roles
- Patient management without patient accounts
- Appointment booking and rescheduling
- Automated SMS and email reminder jobs
- Daily front-desk calendar view
- No-show reporting

## Local setup

1. Install dependencies with npm.
2. Copy the environment example into a local environment file.
3. Run Prisma generate and migrate commands.
4. Start the development server.

## Product principle

Keep v1 focused on front-desk workflows: manage patients, book appointments, send reminders, and measure no-shows.
